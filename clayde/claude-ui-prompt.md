# Frontend Development Task

Please build UI components based on the API documentation provided in `API_DOCUMENTATION.md`.

## Requirements:
- Use React, JavaScript
- Create components for each main endpoint group
- Include form validation
- Add loading states and error handling
- Use Axios/Fetch for API calls
- Use Tailwind CSS for styling (or your preferred CSS framework). Use Material 3 design patterns.
- Build UI in 3 language: Kazakh, Russian, English
- It should be bottom nav application with 4 tabs: Home, Services (There will be category list), Orders, Profile (There will be language settings, avatar change, and other personal settings).
- Use Clean Architecture, SOLID patterns. Make project scalable, maintainable, easy to read.


## Project Structure:
- Components in `/src/components`
- API services in `/src/services`
- Types/Interfaces in `/src/types`

Please start by reviewing the API documentation and propose a component structure.
